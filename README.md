### Installation

Expo - An open source toolchain used to make developing with react native even easier

Why do we use Expo -

- [Expo Learn](https://expo.io/learn)

- [Download the Expo App](https://itunes.apple.com/app/apple-store/id982107779)
- [Expo SDK Camera](https://docs.expo.io/versions/latest/sdk/camera/)

Install expo
```
npm install -g expo-cli
```


 To create a project run the following:
 ```
 expo init SOME_PROJECT
 ```
